# FinanceEra – Personal Finance Tracker & Learning Platform

FinanceEra is a **modern personal finance management platform** that helps users track expenses, manage budgets, and learn financial literacy through interactive tools and educational resources.

The goal of this project is to **empower individuals to take control of their financial life** by combining **finance tracking and financial education in one platform**.

---

## 🚀 Features

* 📊 **Expense & Income Tracking** – Track daily financial transactions easily
* 💳 **Budget Management** – Create and manage monthly budgets
* 📈 **Financial Insights Dashboard** – Visualize spending patterns
* 📚 **Finance Learning Section** – Learn personal finance concepts
* 🔐 **Secure Authentication System** – User login & registration
* 📱 **Responsive UI** – Works smoothly on desktop and mobile

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/jayshayadav/FinaceEra-Personal-Finance-Tracker-and-Learning-Platform.git
```

Go to project folder

```
cd FinanceEra
```

Install dependencies

```
npm install
```

Run the development server

```
npm start
```
